# Exclusão de Dados — Aisha

**Última atualização:** março de 2026

---

## Como solicitar a exclusão dos seus dados

Nos termos da Lei Geral de Proteção de Dados (LGPD — Lei 13.709/2018), você tem o direito de solicitar a exclusão dos seus dados pessoais a qualquer momento.

---

## O que será excluído

Ao solicitar a exclusão, removeremos permanentemente:

| Dado | Onde está armazenado |
|---|---|
| Perfil do usuário (contexto pessoal, idioma, preferências) | Banco de dados Supabase |
| Histórico de sessão (ID da última conversa) | Banco de dados Supabase |
| Lembretes ativos ou pendentes | Banco de dados Supabase |
| Tarefas agendadas ativas | Banco de dados Supabase |
| Estatísticas de uso | Banco de dados Supabase |

---

## O que não podemos excluir diretamente

Alguns dados são processados por terceiros e estão fora do nosso controle direto:

| Dado | Responsável | Como solicitar exclusão |
|---|---|---|
| Histórico de conversas na OpenAI | OpenAI | [privacy.openai.com](https://privacy.openai.com) |
| Mensagens transmitidas pelo WhatsApp | Meta | [facebook.com/help/contact/540977946302970](https://www.facebook.com/help/contact/540977946302970) |
| Dados de análise do Google/Gemini | Google | [myaccount.google.com/data-and-privacy](https://myaccount.google.com/data-and-privacy) |

---

## Como fazer a solicitação

### Opção 1 — Pelo próprio WhatsApp

Envie uma mensagem para a Aisha com o texto:

> **"quero excluir meus dados"** ou **"delete my data"**

A Aisha confirmará a solicitação e encaminhará o pedido.

### Opção 2 — Por e-mail

Envie um e-mail para **contato@askaisha.com.br** com:

- **Assunto:** Solicitação de Exclusão de Dados
- **Corpo:** Seu número de telefone no formato internacional (ex: `+55 85 99999-9999`) e o pedido de exclusão

### Opção 3 — Pela URL de exclusão de dados (exigida pela Meta)

Acesse: **www.askaisha.com.br/data-deletion**

Preencha o formulário com seu número de telefone e confirme a solicitação.

---

## Prazo de atendimento

Sua solicitação será atendida em até **15 dias úteis** a partir do recebimento. Você receberá uma confirmação pelo WhatsApp ou por e-mail quando a exclusão for concluída.
